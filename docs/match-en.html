Match Mode

Play multiple consecutive games to determine the overall champion

Overview

In match mode, multiple games are played consecutively to determine the overall winner. The fixed rounds system ensures fairness while providing strategic, high-level competition.

Match System

Fixed Rounds Mode

Play a specified number of games, with the team winning the most games declared the overall champion in this fair system.

Features

  • All games are played to completion
  • Winner determined by final win count
  • Opportunity for comebacks even when behind
  • Ties result in a draw

How to Start a Match

Command Format

/ctf start <game-name> match <game-count>

Usage Examples

3-Game Match

/ctf start arena1 match 3

Play 3 games with most wins taking the championship

5-Game Match

/ctf start arena1 match 5

Play 5 games with most wins taking the championship

Default Settings

/ctf start arena1 match

Use config.yml setting (default 3 games)

How Matches Work

1. Match Start

Participating players are automatically carried over to the next game. Mid-match joining and leaving is possible.

2. Game Execution

Normal CTF game is played. Win/loss results are counted for each game.

3. Interval

5-second break. Preparation time for the next game.

4. Final Results

After all games, the team with most wins is declared match champion!

Score Display

BossBar Display

During matches, you can check progress on the BossBar at the top of the screen.

  • Game Number: Current game / Total games
  • Current Score: Red X wins, Blue Y wins
  • Progress: Visually displayed with progress bar

Chat Notifications

The following information is displayed at the end of each game:

  • Game win/loss results
  • Current match score
  • Countdown to next game start
  • Overall results when match ends

Currency System During Matches

Currency Carryover

During matches, team currency carries over to the next game.

  • Currency is not reset between games
  • Phase end bonus (50G) is earned each game
  • Manage currency with long-term strategy

Comeback Opportunities

The discount system based on score difference allows losing teams to make comebacks!

  • 1 point behind: 10% discount
  • 2 points behind: 20% discount
  • 3 points behind: 30% discount
  • 4+ points behind: 40% discount

Match Strategy Tips

Currency Management

A strategy of saving currency early and making big investments at crucial moments can be effective.

Fixed Members

Keep members consistent during matches to improve teamwork.

Tactical Changes

Analyze opponent tactics and flexibly change countermeasures each game.

Utilize Discounts

When behind, it's your chance! Use discounts to strengthen equipment and aim for a comeback.
