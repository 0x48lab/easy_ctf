Frequently Asked Questions (FAQ)

Common questions and answers about EasyCTF

Gameplay Related

I captured the flag but didn't score

If your team's flag has been taken by the enemy, you cannot score. First retrieve your own flag, then bring the enemy flag back home.

Check your flag status on the scoreboard.

I can't open the shop

Please check the following:

  • Are you right-clicking the emerald?
  • Are you within 15 blocks of the spawn point?
  • Is the game running (build/combat phase)?

I can't change teams

Team changes are only possible before game start (WAITING phase). Once the game begins, you cannot change teams.

Commands: /ctf team red or /ctf team blue

Items don't disappear when I die

Equipment purchased from the shop (weapons/armor) is kept on death. Consumables (food/ender pearls etc.) are dropped.

Keep Weapons, armor, tools
Drop Consumables, food

I can't use ender pearls

Ender pearls cannot be used while carrying a flag. This is intended to restrict flag carrier movement.

Technical Issues

I can't join the game

Please check the following:

  • Are you already in another game? (Use /ctf leave to leave)
  • Is the game full? (Max 10 players per team)
  • Did you enter the game name correctly?

There's severe lag

Ask the administrator to try the following:

  • Reduce number of concurrent games
  • Increase server RAM
  • Disable unnecessary plugins

Text appears garbled

Check the server's character encoding settings. Language files must be saved in UTF-8 format.

Admin FAQ

Games don't start automatically

Check if minimum player count (default 2) is met. You can adjust min-players in config.yml.

default-game:
  min-players: 2

Can't place flags

Please check the following:

  • Are you looking at the target block? (Up to 100 blocks away)
  • Are world protection settings interfering?
  • Do you have the permission (ctf.admin)?

Settings aren't applied

Most settings require a server restart. They also don't affect games already running.

Match stopped mid-way

Force stop with /ctf stop <game-name> and restart. If there are errors in the log, please report them on GitHub Issues.

Game Design FAQ

What's the recommended arena size?

Recommended by player count:

  • 2v2~4v4: 50×50 blocks
  • 5v5~7v7: 100×100 blocks
  • 8v8~10v10: 150×150 blocks

What should I build in build phase?

Priority order:

  1. Walls around the flag (most important)
  2. Sniper positions (high ground)
  3. Moats or obstacles to prevent invasion
  4. Decoy structures

What are balanced shop prices?

Default settings are already balanced. If changing:

  • Basic equipment should cost 2-3x initial currency
  • Top-tier equipment should require 3-4 kill rewards
  • Consumables should be affordable

Other Questions

I have a feature request

Please post your request on GitHub Issues. We'll consider implementation feasibility.

Issues

I found a bug

Please report on GitHub Issues with the following information:

  • Server version
  • Plugin version
  • Error logs (if any)
  • Steps to reproduce

It conflicts with other plugins

Known conflicts with the following plugins:

  • World protection plugins → Exclude CTF worlds
  • PvP management plugins → Disable during CTF
  • Inventory management → Compatibility patches under consideration

Is the source code available?

Yes, it's available on GitHub. Pull requests are welcome.
