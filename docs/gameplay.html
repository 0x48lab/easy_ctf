遊び方ガイド

EasyCTFの詳しい遊び方を解説します

🎮 基本的な遊び方

EasyCTFは、赤チームと青チームに分かれて戦うキャプチャー・ザ・フラッグゲームです。相手チームの旗を奪い、自陣に持ち帰ることで得点を競います。

基本ルール: 敵の旗（ビーコン）を取得し、自陣の旗がある状態で持ち帰ると1点獲得。先に設定スコアに到達したチームの勝利！

🚀 ゲーム参加の流れ

1

ゲーム一覧を確認

/ctf list

現在実行可能なゲームの一覧が表示されます。状態（待機中/進行中）も確認できます。

2

ゲームに参加

/ctf join <ゲーム名>

自動的に人数の少ないチームに配属されます。友達と同じチームになりたい場合は、事前に相談しておきましょう。

3

ゲーム開始を待つ

最小人数（デフォルト2名）が揃うと30秒のカウントダウンが始まります。この間に準備を整えましょう。

管理者は /ctf start <ゲーム名> で即座に開始できます

⚡ ゲームの3つのフェーズ

フェーズ1: 建築フェーズ（2分）

戦闘の準備をする重要な時間です。

基本情報

  • 目的: 防御施設の建築、戦略的な配置
  • PvP: 無効（攻撃できません）
  • 死亡: 即座にリスポーン、アイテム保持
  • 建築材料: 無限のチームカラーブロック（コンクリート・ガラス）
  • ゲームモード: ADVENTURE/SURVIVAL/CREATIVE（サーバー設定による）
旗とスポーン地点の3×3エリア（高さ無制限）は建築禁止！
ブロック接続システム: ブロックは自チームのビーコンまたは既存のチームブロックに接続している必要があります。接続が切れたブロックは自動的に白色（クォーツブロック）に変化し、どちらのチームでも利用可能になります。

フェーズ2: 戦闘フェーズ（2分）

いよいよ本番！敵の旗を奪い、自陣に持ち帰りましょう。

基本情報

  • 目的: 旗の奪取と防衛
  • PvP: 有効（戦闘可能）
  • 死亡: 10〜20秒後にリスポーン（死亡回数で増加）
  • 旗の取得: 敵のビーコンに1.5ブロック以内に接近
  • 得点条件: 自陣の旗がある状態で敵の旗を持ち帰る
旗を持っている間は発光し、エンダーパールなどの移動アイテムが使用不可に！

フェーズ3: 作戦会議フェーズ（15秒〜1分）

ゲーム結果の確認と次への準備時間です。

フェーズの内容

  • 単独ゲーム: 最終結果と統計の表示（1分）
  • マッチモード: 次のゲームへの準備（15秒）
  • 表示内容: スコア、キル数、MVP等

🚩 旗（フラッグ）システム

旗の取得方法

  • 敵チームのビーコン（旗）に1.5ブロック以内に接近
  • 自動的に旗を取得し、プレイヤーが発光
  • ビーコンとその上の色付きガラスが消える

    • 得点する方法

    1. 敵の旗を持った状態で自陣へ向かう
    2. 自陣の旗が設置されていることを確認
    3. 自陣の旗の3ブロック以内に近づくと自動的に得点

    旗のドロップと回収

    • 旗を持ったまま死亡すると、その場に旗がドロップ
    • 味方が触れると即座に自陣に旗が戻る
    • 敵が触れると旗を奪取
    • 30秒間誰も触れないと自動的に元の位置に戻る
    奈落に落ちて死亡した場合、旗は即座に元の位置に戻ります

    💰 ショップシステム

    基本的な使い方

    1. インベントリ内のエメラルドを右クリック
    2. スポーン地点から15ブロック以内でのみ使用可能
    3. チームメンバー全員で通貨（G）を共有

    通貨の獲得方法

    アクション 獲得G 備考
    ゲーム開始時 50G 初期資金
    敵をキル 10G 基本報酬
    旗持ちをキル 20G 防衛ボーナス
    キルアシスト 5G 5秒以内にダメージを与えた敵が死亡
    旗をキャプチャ 30G 得点時のボーナス
    フェーズ終了 50G 各フェーズ終了時

    割引システム

    スコアで負けているチームには自動的に割引が適用されます：

    • 1点差: 10%割引
    • 2点差: 20%割引
    • 3点差: 30%割引
    • 4点差以上: 40%割引

    購入可能なアイテム

    武器

    • 剣（鉄/ダイヤ/ネザライト）
    • 斧（鉄/ダイヤ/ネザライト）
    • 弓・クロスボウ
    • 盾

    防具

    • 鉄防具セット
    • ダイヤ防具セット
    • ネザライト防具セット
    • エリトラ（高額）

    消耗品

    • エンダーパール
    • 金のリンゴ
    • ポーション各種
    • 矢

    建築ブロック

    • 石・木材・土
    • ガラス・黒曜石
    • TNT・レッドストーン
    • 水バケツ・溶岩バケツ
    アイテムには死亡時の扱いが設定されています：
    • KEEP: 死亡しても保持
    • DROP: 死亡時にドロップ
    • DESTROY: 死亡時に消失

    🔗 ブロック接続システム

    基本ルール

    建築フェーズでブロックを設置する際の重要なシステムです：

    • 接続必須: 新しいブロックは自チームのビーコン（旗）または既存のチームブロックに接続している必要があります
    • 接続の定義: 上下左右前後の6方向のいずれかで隣接していること
    • 斜め接続は無効: 対角線上の接続では有効な接続とみなされません

    切断と中立化

    • 接続チェック: ブロックが破壊されると、周囲のブロックの接続状態が自動的にチェックされます
    • 孤立ブロックの中立化: ビーコンへの接続経路がなくなったブロックは自動的に白色（クォーツブロック）に変化
    • 再利用可能: 中立化されたブロックは、どちらのチームも新たな接続基点として利用可能

    戦略的な活用

    防御建築

    重要な接続ポイントを複数経路で補強し、1箇所破壊されても全体が崩壊しないように

    攻撃戦術

    敵の建築物の「要」となる接続ポイントを狙い、大規模な中立化を引き起こす

    橋の建設

    長い橋を作る際は、複数の支柱で接続を維持し、切断リスクを分散

    ⚔️ 戦闘のコツ

    リスポーンシステム

    • 基本リスポーン時間: 10秒
    • 死亡回数による追加: +2秒/死亡（最大20秒）
    • リスポーン後3秒間の無敵時間（発光状態）
    • 無敵中に攻撃すると即座に解除

    敵陣シールドシステム

    敵チームの色付きブロック（コンクリート・ガラス）の上にいると：

    1. シールドが徐々に減少（25秒で枯渇）
    2. シールドが0になるとダメージを受ける
    3. 自陣に戻るとシールドが回復（10秒で全回復）
    敵陣での長期戦は避け、素早く行動しましょう！

    革防具について

    • チーム識別用の革防具は死亡しても失われません
    • 色は変更できないので、チームの識別に使用
    • 防御力は低いので、ショップで強化を検討

    🏆 マッチモード

    複数のゲームを連続して行い、最多勝利チームを決めるモードです。

    マッチの開始方法

    /ctf start <ゲーム名> match <ゲーム数>

    例: /ctf start arena1 match 5 → 5ゲームのマッチを開始

    マッチの特徴

    • プレイヤーとチーム編成はゲーム間で維持
    • 通貨（G）はゲーム間で引き継ぎ
    • 各ゲーム後15秒の作戦会議時間
    • 途中参加・離脱は不可

    勝利条件

    • 設定したゲーム数を完走
    • 最多勝利チームが優勝
    • 同点の場合: 総ブロック設置数で判定

    📊 画面表示の見方

    ボスバー（画面上部）

    • 現在のフェーズと残り時間
    • 緑色: 建築フェーズ
    • 赤色: 戦闘フェーズ
    • 黄色: 作戦会議フェーズ

    サイドバー（画面右）

    ゲーム中の重要情報を常時表示：

    • ゲーム名: 現在のゲーム
    • スコア: 赤●青● （マッチ時）
    • チーム人数: 赤N 青N
    • マッチ状況: 第Nゲーム（マッチ時）
    • 通貨: NG（チーム共有）

    アクションバー（画面下部中央）

    状況に応じたガイドメッセージ：

    • 「建築して防御を固めよう！」（建築フェーズ）
    • 「敵の旗を奪取せよ！」（戦闘フェーズ）
    • 「自陣に戻れ！」（旗所持時）
    • 「[ショップ使用可能]」（スポーン付近）
    • シールド残量（敵陣ブロック上）

    🎯 戦略のヒント

    建築フェーズの戦略

    • 旗の周りに迷路や高い壁を作って防御
    • 偽の通路で敵を混乱させる
    • 味方の脱出ルートも確保
    • 水や溶岩を効果的に配置

    戦闘フェーズの戦略

    • 攻撃組と防御組で役割分担
    • 旗持ちは単独行動を避ける
    • 敵の旗持ちを優先的に狙う
    • 通貨を稼いで装備を強化

    チームワークのコツ

    • チャットで情報共有
    • ショップ購入は相談して
    • 死亡中は敵の動きを報告
    • 旗の護衛と攻撃のバランス

    ❓ よくある質問

    Q: 途中参加はできますか？

    A: 建築フェーズまでなら可能です。戦闘フェーズ以降は参加できません。

    Q: チームを変更できますか？

    A: ゲーム開始前なら /ctf team [red/blue] で変更可能です（人数差2名以内）。

    Q: 旗が見つからない場合は？

    A: ビーコンの光を探しましょう。また、サイドバーに旗の状態が表示されます。

    Q: ゲームから抜けたい場合は？

    A: /ctf leave で離脱できます。ただし、マッチモード中は離脱できません。
