🎯 ゲームプレイガイド

EasyCTFの基本的なゲームプレイ方法と戦術について説明します。

基本ルール

勝利条件

  • 敵の旗（ビーコン）を奪って自陣に持ち帰る
  • 制限時間内により多く得点したチームが勝利
  • 自陣の旗がある状態でのみキャプチャー可能

チーム構成

🔴 赤チーム

  • 最大10名
  • 赤のコンクリートブロック使用
  • 赤のビーコンを防衛

🔵 青チーム

  • 最大10名
  • 青のコンクリートブロック使用
  • 青のビーコンを防衛

🏗️ 建築フェーズ（2分）

目的: 防衛施設の建設と戦闘準備

できること

  • 自チーム色のコンクリートブロックで防衛施設を建設
  • ショップでアイテムを購入（エメラルド右クリック）
  • マップの偵察（敵陣には侵入不可）
  • チームメンバーとの作戦会議

ブロック接続システム

重要: すべてのブロックは自陣ビーコンまたは既存のチームブロックに接続必須！
  • 接続が切れたブロックは自動的に白色（中立）に変化
  • 敵の建築物の接続を切ることで無効化可能
  • 戦略的な建築配置が重要

建築のコツ

🏰 旗の防衛

  • ビーコン周りを多層構造で囲む
  • 入口を限定して防衛しやすく
  • 高所からの侵入にも注意

🛤️ 進入路制限

  • 迷路状の構造で時間稼ぎ
  • 罠を設置して敵を妨害
  • 複数の防衛ラインを構築

🏹 高所確保

  • 弓での攻撃に有利
  • 全体の戦況把握
  • 旗キャリアの支援

⚔️ 戦闘フェーズ（2分）

目的: 敵の旗を奪って自陣に持ち帰る

旗システム

項目 説明
取得方法 敵ビーコンに1.5ブロック以内で接近
キャリア効果 発光（全員から見える）、移動制限（エンダーパール使用不可）
ドロップ 死亡時にその場にドロップ（30秒で自動復帰）
キャプチャー条件 自陣の旗がある状態で自陣ビーコンに接近

戦闘戦術

攻撃班

2-3名で敵旗の奪取を目指す。エンダーパールで素早く侵入。

防衛班

2-3名で自陣の旗を守る。高所から弓で狙撃。

支援班

状況に応じて攻撃・防衛を支援。旗キャリアの護衛。

リスポーンシステム

即座にリスポーン！ 死亡後の待機時間はありません。すぐに戦線復帰できます。

リスポーン後3秒間のスポーン保護（攻撃または旗取得で解除）

🛡️ シールドシステム

重要: シールドは自陣でのみ回復します！

シールドの仕組み

状態 効果 説明
敵陣ブロック上 毎秒2減少 敵の色のコンクリート上
敵ビーコンエリア 毎秒2減少 敵ビーコン周囲3x3範囲
シールド0 1.5ダメージ/秒 防具で軽減可能
自陣ブロック上 毎秒5回復 自チームの色のコンクリート上
自陣ビーコンエリア 毎秒5回復 + 空腹度回復 自陣ビーコン周囲3x3範囲

警告レベル

  • シールド40以下: 警告メッセージ表示
  • シールド20以下: クリティカル警告
  • シールド0: 継続ダメージ開始

🎁 イベントチェスト

戦闘フェーズ中に1回、特別なチェストが出現します！

特徴

  • 高価なショップアイテムがランダムに入っている
  • 全プレイヤーに出現通知
  • 早い者勝ちで獲得
  • 武器、防具、消耗品がバランスよく出現

💭 作戦会議フェーズ（15秒）

  • ラウンド結果の確認
  • 次ラウンドへの準備
  • 装備の整理
  • チーム戦略の見直し
インターバルなし！ 作戦会議後、すぐに次のフェーズが開始されます。

🎯 Gameplay Guide

Learn the basics of EasyCTF gameplay and tactics.

Basic Rules

Victory Conditions

  • Capture the enemy flag (beacon) and bring it back to your base
  • Team with more points within time limit wins
  • Can only capture when your own flag is present

Team Composition

🔴 Red Team

  • Maximum 10 players
  • Uses red concrete blocks
  • Defends red beacon

🔵 Blue Team

  • Maximum 10 players
  • Uses blue concrete blocks
  • Defends blue beacon

🏗️ Build Phase (2 minutes)

Objective: Build defenses and prepare for combat

What You Can Do

  • Build defensive structures with team-colored concrete blocks
  • Purchase items from shop (right-click emerald)
  • Scout the map (cannot enter enemy territory)
  • Plan strategies with team members

Block Connection System

Important: All blocks must connect to team beacon or existing team blocks!
  • Disconnected blocks automatically turn white (neutral)
  • Can disable enemy structures by cutting connections
  • Strategic building placement is crucial

Building Tips

🏰 Flag Defense

  • Surround beacon with multiple layers
  • Limit entrances for easier defense
  • Watch for aerial invasions

🛤️ Path Restriction

  • Create maze structures to buy time
  • Set traps to hinder enemies
  • Build multiple defense lines

🏹 High Ground

  • Advantageous for bow attacks
  • Overview of battlefield
  • Support flag carriers

⚔️ Combat Phase (2 minutes)

Objective: Capture enemy flag and bring it back

Flag System

Item Description
Capture Method Approach within 1.5 blocks of enemy beacon
Carrier Effects Glowing (visible to all), movement restrictions (no ender pearls)
Drop Drops on death (auto-returns after 30 seconds)
Capture Condition Approach own beacon while own flag is present

Combat Tactics

Attack Squad

2-3 players aim to capture enemy flag. Quick entry with ender pearls.

Defense Squad

2-3 players defend own flag. Snipe from high ground with bows.

Support Squad

Support attack/defense as needed. Escort flag carriers.

Respawn System

Instant Respawn! No waiting time after death. Return to battle immediately.

3 seconds of spawn protection after respawn (canceled by attacking or flag pickup).

🛡️ Shield System

Important: Shield only recovers in your own territory!

Shield Mechanics

State Effect Description
On Enemy Blocks -2 per second On enemy-colored concrete
Enemy Beacon Area -2 per second 3x3 area around enemy beacon
Shield at 0 1.5 damage/sec Can be reduced by armor
On Own Blocks +5 per second On team-colored concrete
Own Beacon Area +5/sec + hunger recovery 3x3 area around own beacon

Warning Levels

  • Shield below 40: Warning message
  • Shield below 20: Critical warning
  • Shield at 0: Continuous damage starts

🎁 Event Chest

A special chest appears once during combat phase!

Features

  • Contains random expensive shop items
  • All players are notified
  • First come, first served
  • Balanced mix of weapons, armor, and consumables

💭 Strategy Phase (15 seconds)

  • Check round results
  • Prepare for next round
  • Organize equipment
  • Review team strategy
No Intervals! Next phase starts immediately after strategy meeting.
