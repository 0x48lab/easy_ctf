Command List

All available commands in EasyCTF

Player Commands

Game Join/Leave

/ctf join <game-name>

Join the specified game. You'll be automatically placed on the team with fewer players.

Example: /ctf join arena1

/ctf leave

Leave the current game. You can leave even during gameplay.

Team Related

/ctf team

Check your current team.

/ctf team <red|blue>

Change teams (only before game starts).

Example: /ctf team red

Game Information

/ctf list

Display all games on the server.

/ctf status [game-name]

Check detailed game status. If game name is omitted, shows current game info.

Example: /ctf status arena1

Admin Commands

These commands require ctf.admin permission.

Game Creation & Management

/ctf create <game-name>

Create a new game (interactive mode).

  1. Set red team flag location
  2. Set red team spawn point
  3. Set blue team flag location
  4. Set blue team spawn point
  5. Select build phase game mode
  6. Set phase durations

Example: /ctf create arena1

/ctf update <game-name>

Update existing game settings (interactive mode). You can select and update individual settings.

Cannot update while game is running.
/ctf delete <game-name>

Delete a game. A confirmation prompt will appear.

Game Start/Stop

/ctf start <game-name> [match] [game-count]

Start a game or match.

Single Game Start:

/ctf start arena1

Match Mode Start:

/ctf start arena1 match 3          # Play 3 games
/ctf start arena1 match 5          # Play 5 games
/ctf start arena1 match            # Default setting (3 games)
Mode Description
first_to_x First team to reach specified wins wins the match
fixed_rounds Play specified number of games
/ctf stop <game-name>

Force stop a running game or match.

Position Settings

/ctf setflag <game-name> <red|blue>

Set flag position to current location. Can also set to the block you're looking at.

Example: /ctf setflag arena1 red

/ctf setspawn <game-name> <red|blue>

Set spawn point to current location.

Must be at least 3 blocks away from the flag.

Example: /ctf setspawn arena1 blue

Permission System

Permission Description Default Included Commands
ctf.use Basic player commands Everyone join, leave, team, list, status
ctf.admin Admin commands OP create, update, delete, start, stop, setflag, setspawn
ctf.* All commands OP All commands
